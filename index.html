Vampire Survivors [v1.9] Save Editor

Last updated: April 18th, 2024 — Source

Upload save file: or paste its content below:

Save file location:

  • Windows: %appdata%\Vampire_Survivors_Data\SaveData.sav
    (which translates to something like C:\Users\<your username>\AppData\Roaming\Vampire_Survivors_Data\SaveData.sav)
    or see this video (though the exact path has slightly changed since the video was made)

  • Windows (XBox): %LocalAppData%\Packages\poncle.VampireSurvivors_9pv5cyp4vwdsr\SystemAppData\wgs\00090000098D240D_00000000000000000000000065116356\{long number foldername}\{long number filename}
    (which translates to something like C:\Users\<your username>\AppData\Local\Packages\poncle.VampireSurvivors_9pv5cyp4vwdsr\SystemAppData\wgs\00090000098D240D_00000000000000000000000065116356\0123456789abcdef0123456789abcdef\0123456789abcdef0123456789abcdef)
    Make sure that VS is closed and your network connection is stopped before editing the file. It gets regenerated and synced with Xbox Live very easily.

  • MacOS: $HOME/Library/Application Support/Vampire_Survivors_Data/SaveData.sav

  • Linux: $HOME/.config/Vampire_Survivors_Data/SaveData.sav

Make sure you make a copy of your save file before editing it in case something goes wrong!

NOTE: You need to press "Update checksum" after editing! Otherwise, the game will think the save is corrupted.

You can ignore any advice you may have read elsewhere about making sure the formatting doesn't change or keeping it all on one line. This editor will automatically make sure the checksum is calculated accurately, and the file is formatted correctly (now).
